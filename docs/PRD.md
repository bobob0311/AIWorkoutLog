# PRD: 헬스 기록 웹사이트

## 목표
달력 중심으로 운동 기록을 빠르게 남기고, 통계와 비교 화면에서 부위별 운동량과 성장 흐름을 쉽게 확인하는 개인용 헬스 기록 웹사이트를 만든다.

## 사용자
- 꾸준히 헬스 기록을 남기고 싶은 개인 사용자
- 날짜 단위로 운동 내용을 빠르게 확인하고 싶은 사용자
- 부위별 볼륨, 주간 추세, 운동별 수행 변화를 보고 다음 운동을 정하고 싶은 사용자

## 핵심 기능
1. 프로필 선택과 비밀번호 기반 간단 로그인
2. 월간 달력에서 날짜별 기록 여부와 부위별 볼륨 확인
3. 날짜 상세 또는 모달에서 운동 기록 생성, 수정, 삭제
4. 기본 운동 목록 검색과 개인 운동 추가
5. 통계 화면에서 기간별 총 볼륨, 세트 수, 부위별 볼륨, 운동 랭킹 확인
6. 비교 화면에서 다음 운동 추천, 지난주 대비, 운동별 최근 세션 비교 확인

## 화면 역할
- 달력: 기록을 시작하고 날짜별 운동 흔적을 보는 화면
- 통계: 선택 기간의 집계와 랭킹을 보는 화면
- 비교: 지난 기록과 비교해 다음 행동을 판단하는 화면

## 성공 기준
- 사용자는 로그인 후 달력에서 운동한 날짜와 총 볼륨을 한눈에 볼 수 있다.
- 사용자는 특정 날짜에서 운동 기록을 생성, 수정, 삭제할 수 있다.
- 사용자는 통계 화면에서 선택 기간의 총 볼륨, 세트 수, 부위별 볼륨, 운동 랭킹을 확인할 수 있다.
- 사용자는 비교 화면에서 추천 부위와 그 근거를 확인할 수 있다.
- 통계와 비교 화면은 기본적으로 실제 오늘 날짜를 기준으로 동작한다.
- 사용자에게 보이는 주요 문구와 문서는 가급적 한글로 작성한다.

## MVP 제외 사항
- 친구 추가, 공유, 소셜 기능
- AI 기반 운동 처방
- 자유 부위 추가
- 목표 관리, PR 관리, 챌린지 기능
