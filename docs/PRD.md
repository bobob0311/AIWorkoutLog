# PRD: 헬스 기록 웹사이트

## 목표
달력 중심으로 운동 기록을 빠르게 남기고, 부위별 운동 볼륨을 일/주/월 기준으로 확인할 수 있는 개인용 헬스 기록 웹사이트를 만든다.

## 사용자
- 꾸준히 운동 기록을 남기고 싶은 개인 사용자
- 날짜 단위로 운동 내용을 돌아보고 싶은 사용자
- 부위별 운동량을 수치로 확인하고 싶은 사용자

## 핵심 기능
1. 프로필 선택 + 비밀번호 기반의 간단한 로그인
2. 월간 달력에서 날짜별 기록 여부와 부위별 볼륨 확인
3. 날짜 상세 페이지에서 운동 기록 CRUD
4. 기본 운동 목록 검색과 개인 운동 추가/재사용
5. 부위별 일/주/월 볼륨 통계 확인

## MVP 제외 사항
- 친구 추가, 공유, 소셜 기능
- 운동 추천, AI 피드백
- 자유 부위 추가 기능
- 목표 관리, PR 관리, 챌린지 기능

## 성공 기준
- 로그인 후 달력 화면에서 운동한 날짜와 부위별 볼륨을 한눈에 볼 수 있다.
- 특정 날짜에서 운동 기록을 생성, 수정, 삭제할 수 있다.
- 기본 운동과 개인 추가 운동을 검색해 재사용할 수 있다.
- 일/주/월 단위 통계가 동일한 볼륨 계산 기준으로 정확히 보여진다.
